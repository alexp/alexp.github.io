teatrdlawas.pl

Frontend, backend and maintenance

Comprehensive vortal all things theatres in Poland.
Performances, dates, news, actors and community.

I was hired to build a brand new version of a theatre oriented vortal.
We've successfully collaborated with the client and his designers on a clean, fast website and a content management system to support all the publishing work done daily by a team of authors.

I delivered a custom solution based on Ruby On Rails framework
and Postgresql database. The website is deployed on a unicorn webserver behind an nginx reverse proxy.

Technologies used:

  • Ruby On Rails
  • Postgreql
  • Full text search
  • Unicorn
  • Nginx
  • jQuery
  • Facebook API for logging in and sharing
  • Omniauth
  • Bluecloth
  • Paperclip
  • Opinio
  • New Relic
